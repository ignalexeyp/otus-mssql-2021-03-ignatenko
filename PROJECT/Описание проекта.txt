Краткое описание системы:
  Единицей проведения маркетинговой кампании - Маркетинговая кампания.
  Маркетинговая кампания состоит из одного или нескольких потоков.
  Для реализации кампании запускаются
потоки. Потоки можно запустить в ручную, при помощи планировщика (встроенного в приложение) или в ответ на
заданный триггер. Данные потоков могут объединяться.
  Потоки состоят из процессов, которые можно соединять и конфигурировать. Процессы бывают различных типов и
выполняют различнве функции - формировании целевой аудитории, воздействие, анализ. Состав типов потоков можно
расширять. Запуск процессов может происходить в ручную, по по таймеру, по триггеру (инициирующее запускпроцесса событие).
   Разработки запроса для формирования целевой аудитории может производиться с помощью самописного генератора запросов
(пользователь не имеющий навыка программирования) или с помощью ранее подготовленных запросоа, хранимых процедур.
   Имеется возможность массовой загрузки данных предприятий и физических лиц.
   Имеется возможность формирования адресного пространства на основе данных ФИАС.
   В БД дипломного проекта не включены таблицы анализа продаж товаров, формирования печатных форм документов,
обеспечения прав доступа пользователей к системе, взаимодействия с виртуальными АТС.
